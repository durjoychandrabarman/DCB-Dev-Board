Question-1: What are the different ways to select an element in the DOM?

  • getElementById() : একটি নির্দিষ্ট id থাকা element ধরতে ব্যবহার হয়
  • getElementsByTagName() একই tag (div, p ইত্যাদি) ধরতে
  • querySelector() CSS selector ব্যবহার করে প্রথম matching element ধরে
  • querySelectorAll() সব matching element ধরে

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML এটি কোনো element এর ভেতরের লেখা HTML সহ দেখায় বা পরিবর্তন করে। HTML ট্যাগ কাজ করে।
  • innerText এটি শুধু যে লেখা স্ক্রিনে দেখা যায়, সেটুকুই দেয় । CSS দিয়ে লুকানো লেখা দেখায় না।
  • textContent : এটি element এর ভেতরের সব লেখা দেয়, এমনকি লুকানো লেখা থাকলেও । HTML ট্যাগ কাজ করে না।/li>

Question-3: What is event delegation in the DOM?

Event Delegation হলো এমন একটি পদ্ধতি যেখানে একাধিক child element এর জন্য আলাদা আলাদা event না দিয়ে, parent element এ একটাই event ব্যবহার করা হয়। এর সুবিধা হলো— কোড কম হয় মেমোরি কম লাগে , নতুন করে যোগ করা element এর উপরও event কাজ করে ।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো এমন একটি প্রক্রিয়া যেখানে , কোনো child element এ event ঘটলে, সেটি ধাপে ধাপে parent element এর দিকে উপরে উঠতে থাকে । অর্থাৎ event এর প্রবাহ হয় — Child → Parent → Body → Document এটাই Event Bubbling।
